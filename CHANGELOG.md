# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-29

### Added
- Initial release of Date Range Selector package
- Customizable date range picker with elegant UI
- Support for custom primary colors
- Configurable dialog title
- Optional onApply callback
- Auto-adjustment of date ranges
- Support for starting with either "from" or "to" date selection

### Features
- Clean, modern UI design
- Responsive calendar picker
- Date validation and range logic
- Easy integration with any Flutter app
- Full customization support
