# Date Range Selector Example

This is an example Flutter application demonstrating how to use the `date_range_selector` package.

## Running the Example

1. Make sure you have Flutter installed
2. Navigate to the example directory:
   ```bash
   cd example
   ```

3. Get dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```

## What This Example Shows

- How to integrate `date_range_selector` in your app
- Custom color theming
- Date range selection and display
- Clearing selected dates
- Formatting dates for display

## Code Highlights

The main implementation can be found in `lib/main.dart`, which demonstrates:

- Importing the package
- Using `showCustomDateRangePicker()` function
- Customizing the primary color
- Setting a custom title
- Handling the date selection callback
- Displaying selected dates in your UI
